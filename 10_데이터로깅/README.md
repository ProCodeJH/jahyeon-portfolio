# 데이터 로깅 프로젝트

## 프로젝트 개요
이 프로젝트는 다양한 센서(온도, 습도, 조도 등)에서 데이터를 수집하고, SD 카드에 저장하는 데이터 로깅 시스템입니다. LCD 화면으로 현재 데이터를 표시하고, 시간 정보와 함께 데이터를 기록합니다.

## 필요 부품
- 아두이노 우노 (Arduino UNO) x1
- DHT11 또는 DHT22 온습도 센서 x1
- 조도 센서(CdS 또는 포토레지스터) x1
- 16x2 LCD 디스플레이 (I2C 모듈 포함) x1
- SD 카드 모듈 x1
- SD 카드 (FAT32 포맷) x1
- RTC(Real Time Clock) 모듈 (DS1307 또는 DS3231) x1
- 10kΩ 저항 x1 (조도 센서용)
- 브레드보드 x1
- 점퍼 와이어 다수

## 회로 연결 방법

### 온습도 센서 (DHT11/DHT22) 연결
1. VCC 핀: 아두이노 5V
2. GND 핀: 아두이노 GND
3. 데이터 핀: 아두이노 디지털 핀 2
4. (선택사항) 데이터 핀과 VCC 사이에 10kΩ 풀업 저항 연결

### 조도 센서 연결
1. 조도 센서 한쪽: 아두이노 5V
2. 조도 센서 다른 쪽: 아두이노 아날로그 핀 A0 및 10kΩ 저항을 통해 GND에 연결
   - 조도 센서 → 아두이노 아날로그 핀 A0
   - 조도 센서 → 10kΩ 저항 → GND

### LCD 디스플레이 (I2C) 연결
1. VCC 핀: 아두이노 5V
2. GND 핀: 아두이노 GND
3. SDA 핀: 아두이노 A4 (SDA)
4. SCL 핀: 아두이노 A5 (SCL)

### SD 카드 모듈 연결
1. VCC 핀: 아두이노 5V
2. GND 핀: 아두이노 GND
3. MISO 핀: 아두이노 디지털 핀 12
4. MOSI 핀: 아두이노 디지털 핀 11
5. SCK 핀: 아두이노 디지털 핀 13
6. CS 핀: 아두이노 디지털 핀 10

### RTC 모듈 (DS1307/DS3231) 연결
1. VCC 핀: 아두이노 5V
2. GND 핀: 아두이노 GND
3. SDA 핀: 아두이노 A4 (SDA)
4. SCL 핀: 아두이노 A5 (SCL)

## 작동 원리
1. 주기적으로 온도, 습도, 조도 데이터를 측정합니다.
2. RTC 모듈에서 현재 시간 정보를 가져옵니다.
3. 측정된 데이터와 시간 정보를 LCD 화면에 표시합니다.
4. 데이터를 CSV 형식으로 SD 카드에 저장합니다.
5. 시리얼 모니터에 현재 데이터와 저장 상태를 표시합니다.
6. 저장된 데이터는 나중에 컴퓨터에서 분석할 수 있습니다.

## 코드 설명
- DHT 라이브러리: 온습도 센서에서 데이터를 읽기 위해 사용
- RTC 라이브러리: 실시간 시간 정보를 얻기 위해 사용
- SD 라이브러리: SD 카드에 데이터를 저장하기 위해 사용
- LiquidCrystal_I2C 라이브러리: I2C LCD 제어를 위해 사용
- 주기적인 측정: millis() 함수를 사용하여 일정 간격으로 데이터 측정
- CSV 형식 데이터 저장: 쉼표로 구분된 형식으로 데이터 저장
- 파일 관리: 새 로그 파일 생성 및 데이터 추가
- 오류 처리: SD 카드 초기화 실패 등의 오류 처리
