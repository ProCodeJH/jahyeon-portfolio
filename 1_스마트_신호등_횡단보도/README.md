# 스마트 신호등 & 횡단보도 버튼 프로젝트

## 프로젝트 개요
이 프로젝트는 실제 도로의 신호등 시스템을 모방한 아두이노 프로젝트입니다. 차량 신호와 보행자 신호를 제어하고, 보행자 요청 버튼을 통해 횡단보도 신호를 변경할 수 있습니다.

## 필요 부품
- 아두이노 우노 (Arduino UNO) x1
- LED (빨강, 노랑, 초록) x3 (차량 신호용)
- LED (빨강, 초록) x2 (보행자 신호용)
- 220Ω 저항 x5
- 푸시 버튼 x1
- 브레드보드 x1
- 점퍼 와이어 다수

## 회로 연결 방법

### 차량 신호등 연결
1. 빨간 LED: 아두이노 디지털 핀 2 → 저항 → LED 양극(긴 다리) → LED 음극(짧은 다리) → GND
2. 노란 LED: 아두이노 디지털 핀 3 → 저항 → LED 양극(긴 다리) → LED 음극(짧은 다리) → GND
3. 초록 LED: 아두이노 디지털 핀 4 → 저항 → LED 양극(긴 다리) → LED 음극(짧은 다리) → GND

### 보행자 신호등 연결
1. 빨간 LED: 아두이노 디지털 핀 5 → 저항 → LED 양극(긴 다리) → LED 음극(짧은 다리) → GND
2. 초록 LED: 아두이노 디지털 핀 6 → 저항 → LED 양극(긴 다리) → LED 음극(짧은 다리) → GND

### 푸시 버튼 연결
1. 푸시 버튼 한쪽 핀: 아두이노 디지털 핀 7
2. 푸시 버튼 다른 쪽 핀: GND
3. 아두이노 디지털 핀 7에 10kΩ 풀업 저항 연결 (핀 7 → 10kΩ 저항 → 5V)

## 작동 원리
1. 기본적으로 차량 신호는 녹색, 보행자 신호는 적색으로 유지됩니다.
2. 보행자가 버튼을 누르면 차량 신호가 노란색으로 변경된 후 적색으로 바뀝니다.
3. 차량 신호가 적색이 되면 보행자 신호가 녹색으로 변경되어 보행자가 길을 건널 수 있게 됩니다.
4. 일정 시간 후 보행자 신호는 깜빡이기 시작하며, 곧 적색으로 변경됩니다.
5. 보행자 신호가 적색으로 변경되면 차량 신호는 녹색으로 변경되어 원래 상태로 돌아갑니다.
