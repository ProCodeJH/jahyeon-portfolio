# 전자 주사위 & 미니게임 프로젝트

## 프로젝트 개요
이 프로젝트는 LED를 사용하여 주사위 눈을 표시하고, 부저를 통해 사운드 효과를 내는 전자 주사위입니다. 버튼을 누르면 주사위가 굴러가며 랜덤한 숫자(1-6)를 표시합니다. 추가적으로 간단한 미니게임 기능도 포함되어 있습니다.

## 필요 부품
- 아두이노 우노 (Arduino UNO) x1
- LED x7 (주사위 표시용 6개 + 추가 게임용 1개)
- 220Ω 저항 x7
- 푸시 버튼 x2 (주사위용 1개, 게임용 1개)
- 부저(Buzzer) x1
- 브레드보드 x1
- 점퍼 와이어 다수

## 회로 연결 방법

### 주사위 LED 연결
주사위 LED는 다음과 같이 배치합니다 (점으로 표시):
```
[1]     [2]
[3] [4] [5] [6]
[7]     [8]
```

실제 연결:
1. LED 1 (중앙 상단): 아두이노 디지털 핀 2 → 저항 → LED 양극 → LED 음극 → GND
2. LED 2 (중앙): 아두이노 디지털 핀 3 → 저항 → LED 양극 → LED 음극 → GND
3. LED 3 (좌측 중간): 아두이노 디지털 핀 4 → 저항 → LED 양극 → LED 음극 → GND
4. LED 4 (중앙 중간): 아두이노 디지털 핀 5 → 저항 → LED 양극 → LED 음극 → GND
5. LED 5 (우측 중간): 아두이노 디지털 핀 6 → 저항 → LED 양극 → LED 음극 → GND
6. LED 6 (좌측 하단): 아두이노 디지털 핀 7 → 저항 → LED 양극 → LED 음극 → GND
7. LED 7 (우측 하단): 아두이노 디지털 핀 8 → 저항 → LED 양극 → LED 음극 → GND

### 게임용 LED 연결
8. 게임 LED: 아두이노 디지털 핀 9 → 저항 → LED 양극 → LED 음극 → GND

### 부저 연결
1. 부저: 아두이노 디지털 핀 10 → 부저 → GND

### 버튼 연결
1. 주사위 버튼: 아두이노 디지털 핀 11 → 버튼 → GND (내부 풀업 저항 사용)
2. 게임 버튼: 아두이노 디지털 핀 12 → 버튼 → GND (내부 풀업 저항 사용)

## 작동 원리
1. 주사위 모드:
   - 주사위 버튼을 누르면 주사위가 굴러가는 효과와 함께 부저에서 사운드가 나옵니다.
   - 랜덤한 숫자(1-6)가 LED로 표시됩니다.
   - 각 숫자에 해당하는 LED 패턴이 표시됩니다.

2. 미니게임 모드:
   - 게임 버튼을 눌러 게임 모드로 전환합니다.
   - 게임 LED가 빠르게 깜빡이며, 특정 패턴에서 멈춥니다.
   - 사용자는 멈춘 패턴을 기억하고 동일한 버튼 조작으로 재현해야 합니다.
   - 성공하면 성공음과 함께 모든 LED가 깜빡이고, 실패하면 실패음이 나옵니다.
