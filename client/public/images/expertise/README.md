# Expertise Images

이 폴더에는 Home 페이지의 Expertise 섹션에 사용되는 배경 이미지들이 들어갑니다.

## 필요한 이미지:

1. **coding-education.jpg** - Coding Education 카드 배경
2. **embedded-systems.jpg** - Embedded Systems 카드 배경
3. **iot-solutions.jpg** - IoT Solutions 카드 배경
4. **software-development.jpg** - Software Development 카드 배경

## 이미지 업로드 방법:

GitHub 웹에서:
1. 이 폴더로 이동
2. "Add file" → "Upload files" 클릭
3. 4개 이미지 파일 드래그 앤 드롭
4. Commit changes

## 권장 사양:

- 포맷: JPG
- 크기: 1920x1080 이상
- 용량: 각 500KB 이하 권장
