# Public Assets

이 폴더에는 웹사이트의 정적 파일들이 들어갑니다.

## 파일 구조:

### `/hero-video.mp4`
- Home 페이지 Hero 섹션의 배경 비디오
- 권장: 5초 분량, 1920x1920 (정사각형), MP4 포맷
- 용량: 5MB 이하 권장

### `/images/expertise/`
- Expertise 카드 배경 이미지 4개
- 자세한 내용은 해당 폴더의 README.md 참조

## 업로드 방법:

GitHub 웹에서:
1. 해당 폴더로 이동
2. "Add file" → "Upload files" 클릭
3. 파일 드래그 앤 드롭
4. Commit changes
