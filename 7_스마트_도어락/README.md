# 스마트 도어락 프로젝트

## 프로젝트 개요
이 프로젝트는 키패드를 사용하여 비밀번호를 입력하고, 서보 모터로 도어락을 제어하는 스마트 도어락 시스템입니다. 올바른 비밀번호가 입력되면 서보 모터가 동작하여 도어락을 열고, 부저를 통해 소리 피드백을 제공합니다.

## 필요 부품
- 아두이노 우노 (Arduino UNO) x1
- 4x4 매트릭스 키패드 x1
- 서보 모터 (SG90 또는 MG996R) x1
- 부저 x1
- 220Ω 저항 x1 (부저용)
- LED (녹색, 적색) x2
- 220Ω 저항 x2 (LED용)
- 브레드보드 x1
- 점퍼 와이어 다수

## 회로 연결 방법

### 4x4 키패드 연결
1. 키패드 왼쪽 위 핀부터 순서대로 아두이노 디지털 핀에 연결:
   - R1: 아두이노 디지털 핀 9
   - R2: 아두이노 디지털 핀 8
   - R3: 아두이노 디지털 핀 7
   - R4: 아두이노 디지털 핀 6
   - C1: 아두이노 디지털 핀 5
   - C2: 아두이노 디지털 핀 4
   - C3: 아두이노 디지털 핀 3
   - C4: 아두이노 디지털 핀 2

### 서보 모터 연결
1. 서보 모터:
   - VCC: 아두이노 5V
   - GND: 아두이노 GND
   - 신호선: 아두이노 디지털 핀 10

### 부저 연결
1. 부저 양극(+) → 220Ω 저항 → 아두이노 디지털 핀 11
2. 부저 음극(-) → 아두이노 GND

### LED 연결
1. 녹색 LED (잠금 해제 상태 표시):
   - LED 양극 → 220Ω 저항 → 아두이노 디지털 핀 12
   - LED 음극 → 아두이노 GND
2. 적색 LED (잠금 상태 표시):
   - LED 양극 → 220Ω 저항 → 아두이노 디지털 핀 13
   - LED 음극 → 아두이노 GND

## 작동 원리
1. 사용자가 4x4 키패드로 비밀번호를 입력합니다.
2. 입력된 비밀번호가 미리 설정된 비밀번호와 일치하는지 확인합니다.
3. 비밀번호가 일치하면 서보 모터를 동작시켜 도어락을 열고, 녹색 LED를 켭니다.
4. 비밀번호가 틀리면 부저를 통해 경고음을 울리고, 적색 LED를 깜빡입니다.
5. 일정 시간 후 자동으로 도어락이 잠깁니다.
6. 시리얼 모니터에 현재 상태와 입력된 비밀번호를 표시합니다.

## 코드 설명
- Keypad 라이브러리: 4x4 키패드 입력을 처리하기 위해 사용
- Servo 라이브러리: 서보 모터 제어를 위해 사용
- 비밀번호 저장: 미리 정의된 비밀번호와 입력값 비교
- 상태 관리: 현재 도어락 상태(잠금/잠금 해제)를 추적
- 피드백 시스템: LED와 부저를 사용하여 사용자에게 시각 및 청각 피드백 제공
- 자동 잠금: 일정 시간 후 자동으로 도어락을 잠그는 기능
