# 서보게이트(미니주차장차단기) 프로젝트

## 프로젝트 개요
이 프로젝트는 서보 모터를 사용하여 미니 주차장 차단기를 제어하는 시스템입니다. RFID 카드나 버튼을 통해 차단기를 개방하고, 일정 시간 후 자동으로 닫히는 기능을 구현합니다. LED 표시등으로 현재 상태를 시각적으로 확인할 수 있습니다.

## 필요 부품
- 아두이노 우노 (Arduino UNO) x1
- 서보 모터(SG90 또는 MG996R) x1
- LED (빨강, 초록) x2
- 220Ω 저항 x2
- 푸시 버튼 x2 (개방용, 닫기용)
- RFID 리더기 모듈 (RC522) x1 (선택 사항)
- 브레드보드 x1
- 점퍼 와이어 다수

## 회로 연결 방법

### 서보 모터 연결
1. 서보 모터 신호선(주로 주황색): 아두이노 디지털 핀 9
2. 서보 모터 전원선(빨간색): 아두이노 5V
3. 서보 모터 접지선(갈색 또는 검은색): 아두이노 GND

※ MG996R과 같은 고토크 서보 모터 사용 시 외부 전원(5V 2A 이상) 연결 권장

### LED 연결
1. 빨간 LED (차단기 닫힘 상태): 아두이노 디지털 핀 2 → 저항 → LED 양극 → LED 음극 → GND
2. 초록 LED (차단기 열림 상태): 아두이노 디지털 핀 3 → 저항 → LED 양극 → LED 음극 → GND

### 버튼 연결
1. 개방 버튼: 아두이노 디지털 핀 4 → 버튼 → GND (내부 풀업 저항 사용)
2. 닫기 버튼: 아두이노 디지털 핀 5 → 버튼 → GND (내부 풀업 저항 사용)

### RFID 리더기 연결 (선택 사항)
1. SDA(SS): 아두이노 디지털 핀 10
2. SCK: 아두이노 디지털 핀 13
3. MOSI: 아두이노 디지털 핀 11
4. MISO: 아두이노 디지털 핀 12
5. RST: 아두이노 디지털 핀 9
6. 3.3V: 아두이노 3.3V
7. GND: 아두이노 GND

※ RFID 모듈 사용 시 서보 모터의 RST 핀은 다른 핀(예: 8)으로 변경해야 합니다.

## 작동 원리
1. 초기 상태에서 차단기는 닫혀있고 빨간 LED가 켜집니다.
2. 개방 버튼을 누르거나 등록된 RFID 카드를 접촉하면 차단기가 열립니다.
3. 차단기가 열리면 초록 LED가 켜집니다.
4. 일정 시간(예: 5초) 후 차단기가 자동으로 닫힙니다.
5. 닫기 버튼을 누르면 차단기가 즉시 닫힙니다.
6. 차단기가 움직일 때 부저 효과음이 나와 상태 변화를 알립니다.
