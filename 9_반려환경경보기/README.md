# 반려환경 경보기 프로젝트

## 프로젝트 개요
이 프로젝트는 PIR(적외선) 움직임 감지 센서를 사용하여 움직임을 감지하고, 알람을 울리는 반려환경 경보 시스템입니다. 다양한 모드(감시 모드, 야간 모드 등)를 지원하며, LCD 화면으로 현재 상태를 표시합니다.

## 필요 부품
- 아두이노 우노 (Arduino UNO) x1
- PIR(적외선) 움직임 감지 센서 x1
- 16x2 LCD 디스플레이 (I2C 모듈 포함) x1
- 부저 x1
- 220Ω 저항 x1 (부저용)
- LED (녹색, 적색) x2
- 220Ω 저항 x2 (LED용)
- 푸시 버튼 x2 (모드 전환용)
- 10kΩ 저항 x2 (버튼용)
- 브레드보드 x1
- 점퍼 와이어 다수

## 회로 연결 방법

### PIR 센서 연결
1. VCC 핀: 아두이노 5V
2. GND 핀: 아두이노 GND
3. OUT 핀: 아두이노 디지털 핀 2

### LCD 디스플레이 (I2C) 연결
1. VCC 핀: 아두이노 5V
2. GND 핀: 아두이노 GND
3. SDA 핀: 아두이노 A4 (SDA)
4. SCL 핀: 아두이노 A5 (SCL)

### 부저 연결
1. 부저 양극(+) → 220Ω 저항 → 아두이노 디지털 핀 8
2. 부저 음극(-) → 아두이노 GND

### LED 연결
1. 녹색 LED (대기 상태 표시):
   - LED 양극 → 220Ω 저항 → 아두이노 디지털 핀 9
   - LED 음극 → 아두이노 GND
2. 적색 LED (경고 상태 표시):
   - LED 양극 → 220Ω 저항 → 아두이노 디지털 핀 10
   - LED 음극 → 아두이노 GND

### 버튼 연결
1. 모드 전환 버튼:
   - 버튼 한쪽: 아두이노 디지털 핀 11
   - 버튼 다른 쪽: 아두이노 5V
   - 아두이노 디지털 핀 11과 GND 사이에 10kΩ 저항 연결 (풀다운)
2. 알람 정지 버튼:
   - 버튼 한쪽: 아두이노 디지털 핀 12
   - 버튼 다른 쪽: 아두이노 5V
   - 아두이노 디지털 핀 12와 GND 사이에 10kΩ 저항 연결 (풀다운)

## 작동 원리
1. PIR 센서를 사용하여 움직임을 감지합니다.
2. 감지된 움직임에 따라 알람을 울리고 LED를 통해 상태를 표시합니다.
3. 다양한 모드를 지원합니다:
   - 감시 모드: 항상 움직임 감지
   - 야간 모드: 어두울 때만 움직임 감지 (조도 센서 필요)
   - 비활성 모드: 움직임 감지 안 함
4. 버튼을 통해 모드를 전환하고 알람을 정지할 수 있습니다.
5. LCD 화면에 현재 모드와 상태를 표시합니다.
6. 시리얼 모니터에 현재 상태와 이벤트 로그를 표시합니다.

## 코드 설명
- LiquidCrystal_I2C 라이브러리: I2C LCD 제어를 위해 사용
- PIR 센서 처리: 움직임 감지 및 디바운싱
- 다중 모드 시스템: 다양한 동작 모드 구현
- 알람 시스템: 움직임 감지 시 알람 울림
- 버튼 입력 처리: 모드 전환 및 알람 정지
- 상태 관리: 현재 시스템 상태 추적 및 표시
