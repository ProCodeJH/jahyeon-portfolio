# 온습도 알람 및 간이 환경 모니터 프로젝트

## 프로젝트 개요
이 프로젝트는 온도와 습도를 측정하여 LCD 화면에 표시하고, 특정 조건(온도나 습도가 설정된 임계값을 초과)에서 알람을 울리는 간이 환경 모니터 시스템입니다. DHT11/DHT22 온습도 센서와 LCD 디스플레이, 부저를 사용하여 구현합니다.

## 필요 부품
- 아두이노 우노 (Arduino UNO) x1
- DHT11 또는 DHT22 온습도 센서 x1
- 16x2 LCD 디스플레이 (I2C 모듈 포함) x1
- 부저 x1
- 220Ω 저항 x1 (부저용)
- 브레드보드 x1
- 점퍼 와이어 다수

## 회로 연결 방법

### 온습도 센서 (DHT11/DHT22) 연결
1. VCC 핀: 아두이노 5V
2. GND 핀: 아두이노 GND
3. 데이터 핀: 아두이노 디지털 핀 2
4. (선택사항) 데이터 핀과 VCC 사이에 10kΩ 풀업 저항 연결

### LCD 디스플레이 (I2C) 연결
1. VCC 핀: 아두이노 5V
2. GND 핀: 아두이노 GND
3. SDA 핀: 아두이노 A4 (SDA)
4. SCL 핀: 아두이노 A5 (SCL)

### 부저 연결
1. 부저 양극(+) → 220Ω 저항 → 아두이노 디지털 핀 8
2. 부저 음극(-) → 아두이노 GND

## 작동 원리
1. DHT11/DHT22 센서를 사용하여 주기적으로 온도와 습도를 측정합니다.
2. 측정된 값을 LCD 화면에 표시합니다.
3. 설정된 온도/습도 임계값을 초과하면 부저를 통해 알람을 울립니다.
4. 버튼을 통해 알람을 일시적으로 정지할 수 있습니다.
5. 시리얼 모니터에 현재 환경 데이터와 시스템 상태를 표시합니다.

## 코드 설명
- DHT 라이브러리: 온습도 센서에서 데이터를 읽기 위해 사용
- LiquidCrystal_I2C 라이브러리: I2C LCD 제어를 위해 사용
- 온도/습도 임계값 설정: 알람이 울리는 조건을 지정
- 주기적인 측정: millis() 함수를 사용하여 일정 간격으로 데이터 측정
- 알람 기능: 임계값 초과 시 부저를 울려 사용자에게 알림
