# 조도 자동등(가로등) 프로젝트

## 프로젝트 개요
이 프로젝트는 조도 센서(CdS 또는 포토레지스터)를 사용하여 주변 밝기를 감지하고, 자동으로 LED를 켜고 끄는 가로등 시스템입니다. 어두워지면 자동으로 불이 켜지고 밝아지면 자동으로 꺼지며, 밝기에 따라 LED 밝기도 조절됩니다.

## 필요 부품
- 아두이노 우노 (Arduino UNO) x1
- 조도 센서(CdS 또는 포토레지스터) x1
- LED x3 (가로등 효과용)
- 220Ω 저항 x3
- 10kΩ 저항 x1 (조도 센서용)
- 브레드보드 x1
- 점퍼 와이어 다수

## 회로 연결 방법

### 조도 센서 연결
1. 조도 센서 한쪽: 아두이노 5V
2. 조도 센서 다른 쪽: 아두이노 아날로그 핀 A0 및 10kΩ 저항을 통해 GND에 연결
   - 조도 센서 → 아두이노 아날로그 핀 A0
   - 조도 센서 → 10kΩ 저항 → GND

### LED 연결
1. LED 1 (가로등 1): 아두이노 디지털 PWM 핀 9 → 저항 → LED 양극 → LED 음극 → GND
2. LED 2 (가로등 2): 아두이노 디지털 PWM 핀 10 → 저항 → LED 양극 → LED 음극 → GND
3. LED 3 (가로등 3): 아두이노 디지털 PWM 핀 11 → 저항 → LED 양극 → LED 음극 → GND

※ PWM 핀(3, 5, 6, 9, 10, 11)을 사용하여 LED 밝기 조절 가능

## 작동 원리
1. 조도 센서가 주변 밝기를 주기적으로 측정합니다.
2. 측정된 밝기 값에 따라 LED의 밝기를 조절합니다.
3. 밝기가 설정된 임계값(예: 300)보다 어두우면 LED가 켜지고, 밝으면 LED가 꺼집니다.
4. 어두운 정도에 따라 LED 밝기를 단계적으로 조절합니다.
   - 매우 어두움: 최대 밝기
   - 중간 어두움: 중간 밝기
   - 약간 어두움: 낮은 밝기
5. 시리얼 모니터에 현재 밝기 값과 LED 상태를 표시합니다.
