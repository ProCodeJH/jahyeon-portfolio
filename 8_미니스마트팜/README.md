# 미니 스마트팜 프로젝트

## 프로젝트 개요
이 프로젝트는 토양 수분 센서를 사용하여 식물의 토양 수분 상태를 감지하고, 자동으로 물을 공급하는 미니 스마트팜 시스템입니다. LCD 화면으로 현재 상태를 표시하고, 릴레이를 사용하여 워터 펌프를 제어합니다.

## 필요 부품
- 아두이노 우노 (Arduino UNO) x1
- 토양 수분 센서 x1
- 16x2 LCD 디스플레이 (I2C 모듈 포함) x1
- 5V 워터 펌프 x1
- 1채널 또는 2채널 릴레이 모듈 x1
- 수조 (물 저장용) x1
- 실리콘 튜브 x1
- 브레드보드 x1
- 점퍼 와이어 다수

## 회로 연결 방법

### 토양 수분 센서 연결
1. VCC 핀: 아두이노 5V
2. GND 핀: 아두이노 GND
3. AOUT (아날로그 출력) 핀: 아두이노 아날로그 핀 A0

### LCD 디스플레이 (I2C) 연결
1. VCC 핀: 아두이노 5V
2. GND 핀: 아두이노 GND
3. SDA 핀: 아두이노 A4 (SDA)
4. SCL 핀: 아두이노 A5 (SCL)

### 릴레이 모듈 연결
1. VCC 핀: 아두이노 5V
2. GND 핀: 아두이노 GND
3. IN 핀: 아두이노 디지털 핀 7

### 워터 펌프 연결 (릴레이 통해)
1. 펌프 전원 (+): 릴레이의 COM (공통) 단자
2. 펌프 전원 (-): 전원 어댑터의 (-) 또는 GND
3. 릴레이의 NO (Normally Open) 단자: 전원 어댑터의 (+)

## 작동 원리
1. 토양 수분 센서를 사용하여 주기적으로 토양의 수분 상태를 측정합니다.
2. 측정된 수분 값이 설정된 임계값보다 낮으면 (토양이 건조하면) 자동으로 펌프를 작동시켜 물을 공급합니다.
3. 물 공급은 설정된 시간 동안만 이루어지며, 그 후 자동으로 멈춥니다.
4. LCD 화면에 현재 토양 수분 상태와 펌프 작동 상태를 표시합니다.
5. 시리얼 모니터에 현재 상태와 시스템 동작 로그를 표시합니다.

## 코드 설명
- LiquidCrystal_I2C 라이브러리: I2C LCD 제어를 위해 사용
- 토양 수분 임계값 설정: 물을 공급할 조건을 지정
- 주기적인 측정: millis() 함수를 사용하여 일정 간격으로 데이터 측정
- 자동 급수: 토양이 건조할 때 펌프를 자동으로 작동
- 상태 표시: LCD와 시리얼 모니터에 현재 상태 표시
- 안전 기능: 펌프 과열 방지를 위해 최대 작동 시간 제한
