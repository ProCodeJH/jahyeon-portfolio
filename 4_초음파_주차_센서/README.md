# 초음파 주차 센서 프로젝트

## 프로젝트 개요
이 프로젝트는 초음파 센서를 사용하여 주차 공간에 차량이 있는지 감지하는 시스템입니다. 차량이 감지되면 LED와 부저를 통해 알림을 제공하며, LCD 화면에 거리 정보를 표시합니다. 주차 공간의 가용 여부를 시각적으로 확인할 수 있습니다.

## 필요 부품
- 아두이노 우노 (Arduino UNO) x1
- 초음파 센서(HC-SR04) x1
- LED (빨강, 초록) x2
- 220Ω 저항 x2
- 부저(Buzzer) x1
- LCD 1602 모듈 (I2C) x1
- 브레드보드 x1
- 점퍼 와이어 다수

## 회로 연결 방법

### 초음파 센서(HC-SR04) 연결
1. VCC: 아두이노 5V
2. Trig: 아두이노 디지털 핀 9
3. Echo: 아두이노 디지털 핀 10
4. GND: 아두이노 GND

### LED 연결
1. 빨간 LED (주차 공간 사용 중): 아두이노 디지털 핀 2 → 저항 → LED 양극 → LED 음극 → GND
2. 초록 LED (주차 공간 비어있음): 아두이노 디지털 핀 3 → 저항 → LED 양극 → LED 음극 → GND

### 부저 연결
1. 부저: 아두이노 디지털 핀 11 → 부저 → GND

### LCD 1602 (I2C) 연결
1. VCC: 아두이노 5V
2. GND: 아두이노 GND
3. SDA: 아두이노 SDA (아두이노 우노: A4)
4. SCL: 아두이노 SCL (아두이노 우노: A5)

## 작동 원리
1. 초음파 센서가 주기적으로 거리를 측정합니다.
2. 측정된 거리가 설정된 임계값(예: 10cm)보다 가까우면 차량이 있는 것으로 판단합니다.
3. 차량이 감지되면 빨간 LED가 켜지고 부저가 울립니다.
4. 차량이 없으면 초록 LED가 켜집니다.
5. LCD 화면에 현재 거리와 주차 공간 상태를 표시합니다.
6. 차량이 감지되면 거리에 따라 부저음의 빠르기가 변합니다 (거리가 가까울수록 빠르게 울림).
