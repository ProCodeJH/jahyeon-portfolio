# 간단한 은행 계좌 관리 시스템

## 프로젝트 개요
이 프로젝트는 은행 계좌를 관리하는 간단한 콘솔 기반 프로그램입니다. 계좌 생성, 조회, 입금, 출금, 이체 등의 기본적인 은행 업무를 수행할 수 있습니다.

## 학습 목표
- 구조체를 사용하여 관련 데이터를 그룹화하는 방법
- 배열을 사용하여 여러 계좌를 저장하고 관리하는 방법
- 함수를 사용하여 코드를 모듈화하는 방법
- 파일 입출력을 사용하여 데이터를 영구적으로 저장하는 방법
- 사용자 입력 유효성 검사 및 오류 처리

## 주요 기능
1. 계좌 생성 (계좌번호, 예금주, 비밀번호, 초기 잔액)
2. 전체 계좌 목록 조회
3. 특정 계좌 정보 조회
4. 입금 기능
5. 출금 기능
6. 계좌 이체
7. 계좌 정보 수정
8. 거래 내역 조회
9. 계좌 정보 파일에 저장 및 불러오기

## 실행 방법
1. 프로그램을 컴파일하고 실행합니다.
2. 메뉴에서 원하는 기능을 선택합니다.
3. 안내에 따라 필요한 정보를 입력합니다.
4. 결과를 확인하고 다른 기능을 계속 사용하거나 프로그램을 종료합니다.

## 코드 구조
- main(): 메인 함수, 프로그램 시작점
- displayMenu(): 메뉴를 표시하는 함수
- createAccount(): 계좌를 생성하는 함수
- displayAllAccounts(): 모든 계좌를 표시하는 함수
- searchAccount(): 계좌를 검색하는 함수
- deposit(): 입금을 처리하는 함수
- withdraw(): 출금을 처리하는 함수
- transfer(): 계좌 이체를 처리하는 함수
- updateAccount(): 계좌 정보를 수정하는 함수
- displayTransactions(): 거래 내역을 표시하는 함수
- saveToFile(): 계좌 정보를 파일에 저장하는 함수
- loadFromFile(): 파일에서 계좌 정보를 불러오는 함수
