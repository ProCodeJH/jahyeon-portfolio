# C언어 프로젝트 예제

이 폴더에는 학생들이 C언어를 학습하고 실습할 수 있는 다양한 프로젝트 예제들이 포함되어 있습니다.

## 프로젝트 목록

1. 성적 관리 시스템
2. 간단한 계산기
3. 연락처 관리 프로그램
4. 텍스트 기반 어드벤처 게임
5. 간단한 은행 계좌 관리 시스템
6. 도서관 관리 시스템
7. 틱택토(Tic-Tac-Toe) 게임
8. 간단한 채팅 프로그램
9. 미니 쇼핑몰 시스템
10. 스네이크 게임

각 프로젝트는 다음과 같은 구조로 되어 있습니다:
- README.md: 프로젝트 개요, 학습 목표, 주요 기능 설명
- 소스 코드 파일: 상세한 주석과 함께 완전한 기능 구현

프로젝트를 통해 학생들은 다음과 같은 C언어 개념을 학습할 수 있습니다:
- 변수, 데이터 타입, 연산자
- 제어문 (if, switch, for, while)
- 배열과 문자열
- 함수와 모듈화
- 포인터와 동적 메모리 할당
- 구조체와 파일 입출력
