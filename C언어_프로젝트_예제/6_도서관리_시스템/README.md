# 도서관리 시스템

## 프로젝트 개요
이 프로젝트는 도서관리 시스템으로서, 도서 정보를 추가, 조회, 수정, 삭제하고 검색하는 기능을 제공하는 콘솔 기반 프로그램입니다. 파일 입출력을 통해 데이터를 영구적으로 저장하고 불러올 수 있습니다.

## 학습 목표
- 구조체를 사용하여 관련 데이터를 그룹화하는 방법
- 배열을 사용하여 여러 도서를 저장하고 관리하는 방법
- 함수를 사용하여 코드를 모듈화하는 방법
- 파일 입출력을 사용하여 데이터를 영구적으로 저장하는 방법
- 문자열 처리 함수를 사용하는 방법
- 검색 알고리즘을 구현하는 방법

## 주요 기능
1. 도서 정보 추가 (도서명, 저자, 출판사, 가격, 분류)
2. 전체 도서 목록 조회
3. 도서 정보 검색 (도서명, 저자, 분류)
4. 도서 정보 수정
5. 도서 정보 삭제
6. 도서 정보 파일에 저장 및 불러오기
7. 도서 대출 및 반납 기능
8. 대출 내역 조회

## 실행 방법
1. 프로그램을 컴파일하고 실행합니다.
2. 메뉴에서 원하는 기능을 선택합니다.
3. 안내에 따라 필요한 정보를 입력합니다.
4. 결과를 확인하고 다른 기능을 계속 사용하거나 프로그램을 종료합니다.

## 코드 구조
- main(): 메인 함수, 프로그램 시작점
- displayMenu(): 메뉴를 표시하는 함수
- addBook(): 도서를 추가하는 함수
- displayAllBooks(): 모든 도서를 표시하는 함수
- searchBook(): 도서를 검색하는 함수
- updateBook(): 도서 정보를 수정하는 함수
- deleteBook(): 도서를 삭제하는 함수
- saveToFile(): 도서 정보를 파일에 저장하는 함수
- loadFromFile(): 파일에서 도서 정보를 불러오는 함수
- borrowBook(): 도서를 대출하는 함수
- returnBook(): 도서를 반납하는 함수
- displayBorrowHistory(): 대출 내역을 표시하는 함수
