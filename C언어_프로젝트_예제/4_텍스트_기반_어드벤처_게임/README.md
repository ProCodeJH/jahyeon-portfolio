# 텍스트 기반 어드벤처 게임

## 프로젝트 개요
이 프로젝트는 텍스트 기반의 간단한 어드벤처 게임입니다. 플레이어는 다양한 방을 탐험하고, 아이템을 수집하며, 퍼즐을 해결하고, 몬스터와 싸워 게임을 진행합니다.

## 학습 목표
- 구조체를 사용하여 게임 데이터를 관리하는 방법
- 함수 포인터를 사용하여 이벤트 처리하는 방법
- 파일 입출력을 사용하여 게임 상태를 저장하고 불러오는 방법
- 랜덤 함수를 사용하여 게임 요소를 생성하는 방법
- 게임 루프를 구현하는 방법

## 주요 기능
1. 플레이어 이동 (북, 남, 동, 서)
2. 방 탐험 및 정보 표시
3. 아이템 수집 및 사용
4. 몬스터와 전투
5. 퍼즐 해결
6. 게임 저장 및 불러오기
7. 인벤토리 관리
8. 게임 클리어 조건 확인

## 실행 방법
1. 프로그램을 컴파일하고 실행합니다.
2. 게임 시작 시 플레이어 이름을 입력합니다.
3. 명령어를 사용하여 게임을 진행합니다.
4. 모든 퍼즐을 해결하고 보스를 물리쳐 게임을 클리어합니다.

## 코드 구조
- main(): 메인 함수, 게임 루프 포함
- initGame(): 게임 초기화 함수
- displayRoom(): 현재 방 정보 표시 함수
- movePlayer(): 플레이어 이동 함수
- handleCombat(): 전투 처리 함수
- handlePuzzle(): 퍼즐 처리 함수
- handleItem(): 아이템 처리 함수
- saveGame(): 게임 저장 함수
- loadGame(): 게임 불러오기 함수
- displayInventory(): 인벤토리 표시 함수
- checkWinCondition(): 승리 조건 확인 함수
