# 성적 관리 시스템

## 프로젝트 개요
이 프로젝트는 학생들의 성적을 관리하는 간단한 콘솔 기반 프로그램입니다. 학생 정보를 입력하고, 성적을 계산하며, 전체 학생들의 성적을 조회하고 정렬하는 기능을 제공합니다.

## 학습 목표
- 구조체를 사용하여 관련 데이터를 그룹화하는 방법
- 배열을 사용하여 여러 데이터를 저장하고 관리하는 방법
- 함수를 사용하여 코드를 모듈화하는 방법
- 파일 입출력을 사용하여 데이터를 영구적으로 저장하는 방법
- 정렬 알고리즘을 구현하는 방법

## 주요 기능
1. 학생 정보 입력 (이름, 학번, 국어, 영어, 수학 점수)
2. 학생 성적 계산 (총점, 평균, 등급)
3. 전체 학생 목록 조회
4. 학생 성적순 정렬
5. 학생 정보 파일에 저장 및 불러오기
6. 특정 학생 정보 검색
7. 학생 정보 수정 및 삭제

## 실행 방법
1. 프로그램을 컴파일하고 실행합니다.
2. 메뉴에서 원하는 기능을 선택합니다.
3. 안내에 따라 필요한 정보를 입력합니다.
4. 결과를 확인하고 다른 기능을 계속 사용하거나 프로그램을 종료합니다.

## 코드 구조
- main(): 메인 함수, 프로그램 시작점
- displayMenu(): 메뉴를 표시하는 함수
- addStudent(): 학생 정보를 추가하는 함수
- displayAllStudents(): 모든 학생 정보를 표시하는 함수
- calculateGrade(): 평균 점수에 따른 등급을 계산하는 함수
- sortStudents(): 학생들을 성적순으로 정렬하는 함수
- saveToFile(): 학생 정보를 파일에 저장하는 함수
- loadFromFile(): 파일에서 학생 정보를 불러오는 함수
- searchStudent(): 학생을 검색하는 함수
- updateStudent(): 학생 정보를 수정하는 함수
- deleteStudent(): 학생 정보를 삭제하는 함수
