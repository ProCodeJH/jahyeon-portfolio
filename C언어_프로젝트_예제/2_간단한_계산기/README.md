# 간단한 계산기

## 프로젝트 개요
이 프로젝트는 기본적인 사칙연산(덧셈, 뺄셈, 곱셈, 나눗셈)을 수행하는 콘솔 기반 계산기 프로그램입니다. 사용자가 두 숫자와 연산자를 입력하면 결과를 계산하여 출력합니다.

## 학습 목표
- 사용자 입력 처리 및 유효성 검사
- 조건문을 사용한 연산자 분기 처리
- 함수를 사용한 코드 모듈화
- 오류 처리 및 예외 상황 대응
- 반복문을 사용한 프로그램 지속 실행

## 주요 기능
1. 기본 사칙연산 수행 (덧셈, 뺄셈, 곱셈, 나눗셈)
2. 사용자 입력 유효성 검사
3. 0으로 나누기 오류 처리
4. 계산 기록 및 조회
5. 연속 계산 기능
6. 소수점 연산 지원

## 실행 방법
1. 프로그램을 컴파일하고 실행합니다.
2. 첫 번째 숫자를 입력합니다.
3. 연산자(+, -, *, /)를 입력합니다.
4. 두 번째 숫자를 입력합니다.
5. 계산 결과를 확인합니다.
6. 계속 계산하거나 프로그램을 종료합니다.

## 코드 구조
- main(): 메인 함수, 프로그램 시작점
- displayMenu(): 메뉴를 표시하는 함수
- getNumber(): 숫자 입력을 받는 함수
- getOperator(): 연산자 입력을 받는 함수
- calculate(): 계산을 수행하는 함수
- addToHistory(): 계산 기록을 추가하는 함수
- displayHistory(): 계산 기록을 표시하는 함수
