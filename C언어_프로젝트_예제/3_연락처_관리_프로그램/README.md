# 연락처 관리 프로그램

## 프로젝트 개요
이 프로젝트는 연락처 정보를 추가, 조회, 수정, 삭제하고 검색하는 기능을 제공하는 콘솔 기반 연락처 관리 프로그램입니다. 파일 입출력을 통해 데이터를 영구적으로 저장하고 불러올 수 있습니다.

## 학습 목표
- 구조체를 사용하여 관련 데이터를 그룹화하는 방법
- 동적 메모리 할당을 사용하여 데이터를 관리하는 방법
- 파일 입출력을 사용하여 데이터를 영구적으로 저장하는 방법
- 문자열 처리 함수를 사용하는 방법
- 검색 알고리즘을 구현하는 방법

## 주요 기능
1. 연락처 정보 추가 (이름, 전화번호, 이메일, 주소)
2. 전체 연락처 목록 조회
3. 연락처 정보 검색 (이름 또는 전화번호)
4. 연락처 정보 수정
5. 연락처 정보 삭제
6. 연락처 정보 파일에 저장 및 불러오기
7. 연락처 정렬 (이름순)

## 실행 방법
1. 프로그램을 컴파일하고 실행합니다.
2. 메뉴에서 원하는 기능을 선택합니다.
3. 안내에 따라 필요한 정보를 입력합니다.
4. 결과를 확인하고 다른 기능을 계속 사용하거나 프로그램을 종료합니다.

## 코드 구조
- main(): 메인 함수, 프로그램 시작점
- displayMenu(): 메뉴를 표시하는 함수
- addContact(): 연락처를 추가하는 함수
- displayAllContacts(): 모든 연락처를 표시하는 함수
- searchContact(): 연락처를 검색하는 함수
- updateContact(): 연락처 정보를 수정하는 함수
- deleteContact(): 연락처를 삭제하는 함수
- sortContacts(): 연락처를 정렬하는 함수
- saveToFile(): 연락처 정보를 파일에 저장하는 함수
- loadFromFile(): 파일에서 연락처 정보를 불러오는 함수
